Univerzalna podrška
-Svi moderni web pretraživači podržavaju Javascript sa ugrađenim prevodiocem.
Imperativan i strukturiran
-Javascript podržava većinu strukturirane sintakse programskog jezika C (npr. If naredbe, while petlje, switch naredbe, do while petlje, itd.).
Delimični izuzetak je opseg varijabli: Javaskript je izvorno podržan samo funkcionalnim opsegom varijabli pomoću var deklaracije. EKMASKript 2015 godinama obezbeđuje ključnu reč za blok band, tako da Javaskript sada podržava funkcionalni i blokirajući opseg.
Kao i C, Javascript pravi razliku između iskaza i izraza. Druga važna razlika u odnosu na C je automatsko umetanje kraja naredbe ili point-to-point karaktera, tako da nije potrebno eksplicitno naglasiti kraj naredbe znakom tačka-zarez(;)

Dinamično
Kucanje
-Kao i većina skriptnih jezika, javascript je jezik sa dinamičkom proverom kucanja; Tip se odnosi na svaku vrijednost prije svakog termina.
Na primer, ako imamo varijablu povezanu s brojem, ista varijabla se kasnije može povezati s nizom. Javascript podržava različite tipove provere tipova objekata.
Verifikacija tokom izvršavanja
-Javascript sadrži deo koji može izvoditi izjave u obliku nizova u vreme izvođenja.
Prototip (objektno orijentisan)
Javascript je gotovo potpuno objektni skriptni jezik. U JavaScriptu, objekt predstavlja asocijativni niz, naglašen prototipom; svaki ključ u obliku niza znakova predstavlja svojstvo objekta objekta za koje postoje dvije metode sintakse za definisanje takvog imena;
notacija sa tačkom, engleski. notna oznaka (obj.x = 10) i notacija s zagradama, engl. oznaka zagrada (obj ['x'] = 10).
Svojstvo se može dodati ili izbrisati tokom izvršavanja koda. Većina svojstava objekta (ili bilo koje imovine koja pripada lancu nasleđivanja prototipskog objekta) može se navesti pomoću ... i petlji. Javascript sadrži ograničen broj ugrađenih objekata kao što su Date i Function.
