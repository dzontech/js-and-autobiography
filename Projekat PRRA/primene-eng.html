Web Programming
The basic application of the Javascript programming language is web programming. Before a browser loads a page, it must create a DOM and a CSSOM tree The browser analyzes the file, it actually analyzes bytes and real characters, and then creates a character from node node and from node to object.
HTML tags are transformed into Object Document Model (DOM) and CSS tags in CSS Object Model (CSSOM). DOM and CSSOM are independent structures and their merger created a rendering tree, which is then used to calculate the properties of each visible element, e.g. the size of the element, the background color, etc.
Optimizing each of these steps is crucial to achieving optimal viewing performance. The DOM describes the content, and CSSOM describes the style used. The rendering tree contains only the nodes needed to display the page and takes care of the appearance and exact size of each object.
When creating a rendering tree, the browser does the following:
- Moves from the root of the DOM tree and visits every visible node - Some nodes are not visible (e.g., Tag script, meta tag, etc.) and do not display them in the DOM tree because they are not visible on the page.
- Some nodes are hidden via CSS (display: none) and are not displayed in the DOM tree
-For every visible node, the search engine finds the appropriate CSSOM rules and applies them.


- Shows visible nodes with content and stylizes them Use on web pages
The most common use of Javascript is the addition of the HTML code dynamics on the client side. The scripts are included in the HTML page and interact with the Object Model of the document Some examples of using Javascript are:
-Setting a specific piece of site or sending data to a server via AJAX (AJAX) without re-updating the whole page (for example, the social network allows you to update the new status without updating the entire page). -Animation of the elements on the page, making them visible and invisible, changes the size of the elements, moves their content. - Reading interactive content (such as video clips, animations, video games, audio content ...) - Confirm the content to make sure the correct data is sent to the server. -The Internet browser is the most common environment for executing Javascript code. Internet browsers typically create host objects for presenting the object model of documents in Javascript. In addition to the Internet browser, the Javascript code can also be executed on the server.
