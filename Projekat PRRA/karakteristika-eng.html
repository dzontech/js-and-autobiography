Universal Support
         -All modern web browsers support Javascript with built-in interpreter
         Imperative and structured
         -Javascript supports most structured syntax of C programming language (for example, If commands, while loops, switch commands, to while loops, etc.). The partial exception is the range of variables:The Javascript is originally supported only by the functional range of variables using the var declaration. EKMASKript for 2015 provides a keyword for the block band, so Javascript now supports the functional and blocking range. Like C, Javascript makes the difference between statements and expressions.          Another important difference in relation to C is the automatic insertion of the end of the command or point-to-point          characters, so it is not necessary to explicitly emphasize the end of the command with a dot-comma (;) sign.

Dynamic
        Typing
        Like most scripting languages, javascript is a dynamic typing language; The type refers to each value before each term.
        For example, if we have a variable associated with a number, the same variable can later be associated with a string. Javascript supports different types of object type checking Verification during execution -Javascript contains a part that can perform statements in the form of arrays at run time
          Prototype (Object Oriented)
Javascript is almost a completely object-oriented scripting language. In JavaScript, the object represents an associative array, emphasized by a prototype; each key in the form of a string of characters represents the property of an object object for which there are two methods of syntax for defining such a name:
        -notation with a dot, English. notation (obj.x = 10) and notation with brackets, engl. bracket label (obj ['x'] = 10).
        The property can be added or deleted during execution of the code. Most properties of an object (or any property belonging to a chain of inheritance of a prototype object) can be specified using ... and loops. Javascript contains a limited number of embedded objects such as Date and Function.
