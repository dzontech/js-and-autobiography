        After that, another search engine was designed. The internal name for this web browser was "Mozilla", which meant "mosaic killer". Mosaic killers, because the company's goal was to replace Mozaik as the world's most popular web browser      The first version of the web browser, Mosaic NetSpace 0.9, was released in late 1994. In just four months, this web browser has already won three quarters of the Internet search market and has become the most popular web browser in the nineties of the twentieth century.
